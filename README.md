# Movie App
Sample App Using The Movie Database API

This app uses:
- Android Native
- Kotlin, Coroutines, Extensions
- MVVM Architecture
- LiveData, Lifecycle, ViewModel, Data Binding
- ConstrainLayout
- Retrofit & OkHttp
- Dagger 2
- Glide
- EventBus
- LeakCanary
- Lottie

For sample only. Will be continuously edited/changed.
